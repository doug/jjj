# Changelog

## [0.1.0] - 2026-03-10

### Added
- Project tree view with milestones, problems, solutions, and critiques
- Inline code critique comment threads with fuzzy re-anchoring
- Solution lifecycle management (propose, submit, approve, withdraw)
- Status bar showing active solution and project summary counts
- Drag-and-drop problems between milestones
- GitHub sync: import issues, create PRs, sync reviews
- Tab/Shift+Tab navigation between open items
- Filter toggle: show all items or open only
- Virtual document viewer for entity details
- Context menu actions for all entity types
- Keyboard shortcut Cmd+Shift+O for filter toggle
